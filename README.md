# Pipeline-para-Analizar-datos-sinteticos
Este proyecto es un Pipeline para analizar datos sintéticos de alumnos de la Universidad Nacional del Altiplano de la ciudad de Puno el cual sirve para analizar el por que la obtención de sus notas finales que esta como x11 usando dataset sintético con 11 variables de x1 a x11, el análisis también se da a través de gráficos
